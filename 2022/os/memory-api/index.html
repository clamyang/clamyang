忘记初始化刚分配的内存

忘记释放内存

分配和释放应该成对的出现，这就好像 Go 中打开了一个 response.body 要记得关闭一个道理。如果是一个长时间运行的服务，最终可能把所有的内存耗光，导致 OOM。如果是一个一次性运行的服务，不是放也没什么关系，进程退出了自动会把占有的资源释放了。
